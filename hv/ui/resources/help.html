Controls programm for Mantigora HV device

Общий обзор

  • Текущие значения тока и напряжения отображаются на индикаторе, обновляемом каждые 2 секунды. Кнопка Start record включает запись показаний индикатора в файл, в формате: время, напряжение, ток.
  • Под индикатором расположены кнопки для выставления напряжение. Доступный два режима: выставление фиксированного значения и генерация периодического сигнала (описание доступных сигналов смотри в следующем разделе)
  • Справа расположены графики напряжения и тока. Кнопка Stop останавливает построение графика. Кнопка Save buffer сохраняет в файл точки графика за последние 10 минут. Так же есть кнопки для сохраненния графика как изображения и для настройки графика.

Описание генераторов

После нажатии кнопки Swith to generator mode, можно выбрать тип периодического сигнала и указать его параметры. Достпны следующие виды сигналов:
  • Square wave — прямоугольные импульсы, можно задать минимальное и максимальное напряжение, предельный ток, период сигнала и его коэффициент заполнения.
  • Stairs — прямая лесенка, напряжение изменяется от минимального до максимального, фиксированны шагами через заданные промежутки времени (с учетом предельного тока).
  • Reversed rawtooth — обратная пила, выставляется максимальное напряжение и линейно спадает до минимального. Можно задать минимальное и максимальное напряжение, предельный ток, период сигнала и его коэффициент заполнения.
  • Custom — использует код из пользовательского файла для генерации периодического сигнала. Кнопка Export template позволяет получить шаблон для кода. В коде нужно определить период сигнала через переменную PERIOD, шаг временной сетки через переменную TIME_STEP, и функцию generator, принимающую на вход время и возвращающую значения напряджения и тока, которые надо выставить на источник.
